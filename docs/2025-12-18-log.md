# Session Log - 2025-12-18

## Project: Server Dashboard

### Goal
Create a real-time server monitoring dashboard similar to https://clopus.live/status for deployment on a Coolify server at `100.64.156.119:8000`.

### Tech Stack Decision
- **Backend**: Go with gopsutil library for system metrics
- **Frontend**: Plain HTML/CSS/JavaScript (no framework)
- **Deployment**: Docker container on Coolify

### Files Created

| File | Purpose |
|------|---------|
| `main.go` | Go HTTP server with `/api/stats` endpoint that collects system metrics (CPU, memory, disk, network, uptime, load averages) using gopsutil |
| `static/index.html` | Dashboard UI with glassmorphism design, auto-refreshes every 5 seconds |
| `go.mod` | Go module definition with gopsutil v3.24.1 dependency |
| `Dockerfile` | Multi-stage build for minimal container (~15MB) |
| `.gitignore` | Exclude build artifacts |

### Deployment Instructions

1. Push repo to GitHub
2. In Coolify (`http://100.64.156.119:8000`):
   - Create new project/application
   - Select "Docker" as build type
   - Connect GitHub repo
   - Set port to 3000
   - Deploy
